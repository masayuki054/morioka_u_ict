1. Doing 受講の準備

講義を受けるために下記のことを行ないましょう:

  • [ ] パソコンにログインし，
  • [ ] パソコン画面上の「出席票」に名前と学生番号を入力してください。
  • [ ] 先生のパソコンの画面が配信されていることを確認し，ウィンドウの大 きさを見易い大きさに変更してください。
  • [ ] (インターネット) ブラウザを立ちあげ，
    • Edge か Chrome か Firefox
    • ブラウザのアカウントにサインインし， 履歴や保存したリンクを使えるようにする
  • [ ] 講義の入口ページを開き，
  • [ ] 別のタブで 今日の講義のページを開いてください。

1.1. Doing OneDrive を開く

以下の内容には，未確認のことがあります。異なる症状があれば，講師に 報告お願いします。

エクスプローラ (ウィンドウズキー+e) から開く

  • OneDrive をクリックして，OneDrive にサインインし, 自分の情報処理演 習フォルダ開きましょう。
  • OneDrive がエクスプローラのサイドバーに現われていない時は， しばらく待ってみてください。それでも現われない場合は，PCを再起動し てください。

ブラウザで OneDrive を開く

  • 再起動しても，OneDrive がエクスプローラのサイドバーに現われていない時は，
    • ブラウザで office 365 にサインインし，OneDrive を開いてください。
    • OneDrive 上の書類をダブルクリックして開くと，office365のWebアプリ が起動します。Webアプリは，デスクトップ版に較べ，機能が低いことが あります。
    • 機能が低い場合，
      • ブラウザの画面から，ダウンロードやアップロードし，デスクトッ プアプリを起動してください。
      • あるいは，書類を右クリックして，デスクトップアプリで開いてく ださい

デスクトップ・アプリで OneDrive の書類を開いた場合

  • OneDriveの書類を直接デスクトップ・アプリで開いた場合，
  • OneDrive からダウンロードした書類をデスクトップ・アプリで開いた場合，

いずれの場合も，デスクトップ・アプリの起動時， 画面上部にメッセージ とボタンが現われると思います。読み取り専用モードになっているので， かならず，編集モードにしてくだい。そうしないと，ファイルの保存時に， アプリがハングアップします。
