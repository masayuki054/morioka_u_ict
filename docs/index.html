ことが大切です。

USB3.0 4G以上空きがあるモノを用意してください。

持っていない人は，盛大生協においてある プリンストン PSUSB3-8G 710円 がおすすめです。

今さら聞けない! USBの基礎知識『USB2.0と3.0の違いと見わけ方』 を参考 にしてください．

退室時の注意
  • パソコンの電源を切るか，次に使う人がいる時はログオフしてください。
  • 出席カードは，先生に手渡ししてください。
